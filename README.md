# 🌍 MixUpProject

포스트 아포칼립스 세계관을 배경으로 하는 오픈월드 생존 게임입니다. 혹독한 환경에서 자원을 채집하고, 피난처를 건설하며, 돌연변이와 위험 요소들 사이에서 생존해야 합니다.

## 🎮 주요 특징 (Features)
- **오픈월드 탐험:** 폐허가 된 도시와 변화하는 기후가 있는 광활한 세계
- **하드코어 생존:** 허기, 갈증, 체온, 피로도 등 세밀한 생존 스탯 관리
- **크래프팅 & 빌딩:** 채집한 자원을 활용한 아이템 제작 및 자신만의 방어 기지(쉘터) 건설
- **위협적인 환경:** 밤이 되면 강력해지는 돌연변이 및 다른 생존자 NPC와의 상호작용

## 🛠 개발 환경 (Tech Stack)
- **Game Engine:** Unreal Engine
- **AI & Behavior:** 언리얼 엔진 비헤이비어 트리 및 커스텀 AI 모듈
- **Version Control:** Git

## 🚀 시작하기 (Getting Started)
1. 레포지토리를 클론합니다.
   ```bash
   git clone <repository-url>
   ```
2. 언리얼 엔진에서 `UEProject` 폴더 내의 `.uproject` 파일을 실행합니다.
3. 기여를 원하시면 `CONTRIBUTING.md` 파일을 참고해 주세요.
